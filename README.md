Soil Moisture Monitoring System

Overview
This project is an IoT-based Soil Moisture Monitoring System developed using Arduino/NodeMCU and a soil moisture sensor. The system measures the moisture level present in the soil and helps in monitoring plant watering conditions. It is mainly used in smart agriculture and automatic irrigation systems to improve water management and plant health.

Features
Real-time soil moisture monitoring
Displays moisture level continuously
Helps in smart irrigation management
Easy and low-cost implementation
Beginner-friendly IoT project
Can be integrated with automatic watering systems

Components Required
Arduino Uno / NodeMCU ESP8266
Soil Moisture Sensor
Breadboard
Jumper Wires
USB Cable
Water Pump (optional)
Relay Module (optional)

Software Used
Arduino IDE
ESP8266 Board Package (if using NodeMCU)
Embedded C Programming

Working Principle
The soil moisture sensor detects the amount of water present in the soil. The sensor sends analog data to the Arduino or NodeMCU. Based on the moisture level, the system displays whether the soil is dry, moist, or wet. The project can also be extended to automatically turn ON a water pump when the soil becomes dry.

Arduino IDE Setup
Open Arduino IDE.
Select the correct board and COM port.
Write or paste the soil moisture monitoring code.
Upload the code to the Arduino/NodeMCU.
Open Serial Monitor to view readings.

Output
Displays soil moisture values in real time
Indicates dry, normal, or wet soil condition
Helps monitor watering needs of plants
